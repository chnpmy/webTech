1. What are JavaScript browser sniffers?


    JavaScript browser sniffers is a kind of browser sniffing technology. It's very important for our browsers because the early browsers use different DOMs and the same javascript code running on one may not running on another. So, we need to use some special Javascript code to find out what the current browser is(Example: IE) and then execute different code according to the result, which can let the same code works well in all browsers.

  2. Why are they so commonly used in javascripts?


    There are some reasons that why browser sniffers are so commonly used in javascripts:
    • Javascript are supported by most browsers.
    • Javascript can be used in HTML page easily.
    • Javascript have some useful methods which can know what current browser are.
