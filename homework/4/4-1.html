1. Give the SELECT command to answer the question Who will go to the party held in Peking University?

    Answer : The SELECT command is SELECT guest.Guest_ID, guest.Guest_Name, guest.E_mail, party.Party_Num, party.Time, party.Place FROM guest, party, guest_party WHERE party.Party_Num=guest_party.Party_Num AND guest_party.Guest_ID=guest.Guest_ID AND party.Place="Peking University";.

  2. Figure out a way to obtain the column labels returned by mysql_fetch_array.

    Answer : The function mysql_fetch_array() has a form like mysql_fetch_array(data, array_type), if you let array_type = "MYSQL_ASSOC", the function return the data as an associated array. If you travel the keys of the array, you will get the column labels.

  3. Tell the difference between a CGI program and a PHP program executed by a Web Server.

    Answer:

    • CGI is separate from web page files, while php can be within web page source code files.
    • Most important thing : php is a kind of language, while CGI means Common Gateway Interface, a kind of technology, They belong to two different levels. In other words, php can be used to implement CGI.
